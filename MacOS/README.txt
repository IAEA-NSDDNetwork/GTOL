 How to run ENSDF programs on MacOSX

Using a Terminal window type the program name space ensdf filename and then press RETURN/ENTER

program_name    ensdf_filename

or for an interactive session just type the program name and then press RETURN/ENTER

Note that the program gtol runs in two different modes:

Mode 1: to obtain the level energies by fitting the gamma-ray energies

gtol ensdf_name : F   and then press RETURN/ENTER

Mode2: to obtain intensity balance in a decay data set:

gtol ensdf_file_name or just gtol followed by RETURN/ENTER
